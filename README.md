This mobile app was made using .NET MAUI framework, with C# and XAML for the functionalities and UI, respectively, and SQLite for the database.


## Functionalities
### The app allows the user to:
- View different types of makeup artists or services
- Book an appointment for a makeup artist
- Leave a review after the appointment
- View other reviews for a makeup artist
- View their appointments
- Choose from a collection of tutorials to watch
